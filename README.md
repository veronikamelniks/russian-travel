# Проект: Путешествие по России
Третий проект курса "Яндекс.Практикум" о путешествиях по России. Адаптивный сайт

**Используемые технологии:**

- Языки HTML и CSS
- Flexbox Верстка
- Grid Layout
- Медиа запросы для корректного отображения на всех устройствах
